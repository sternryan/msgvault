2. Create `internal/web/sanitize_email.go` with:

   a. `newEmailPolicy() *bluemonday.Policy` — custom email HTML policy:
      - `p := bluemonday.NewPolicy()`
      - `p.AllowUnsafe(true)` — required for style blocks; iframe sandbox is the XSS defense
      - AllowElements: html, head, body, div, span, p, br, hr, h1-h6, ul, ol, li, dl, dt, dd, blockquote, pre, code, b, strong, i, em, u, s, strike, sup, sub, small, big, center
      - Table elements: table, thead, tbody, tfoot, tr, td, th, caption, colgroup, col
      - Font element with color, face, size attributes
      - `p.AllowAttrs("style").Globally()` — inline styles for email layout
      - `p.AllowStandardAttributes()` — id, title, dir, lang
      - `p.AllowStandardURLs()`
      - `p.AllowAttrs("href", "name").OnElements("a")`
      - `p.AddTargetBlankToFullyQualifiedLinks(true)` — auto-injects target=_blank + rel=noopener
      - `p.AllowImages()` + `p.AllowDataURIImages()`
      - Table attrs: colspan, rowspan on td/th; align, valign, bgcolor, width, height, border, cellpadding, cellspacing on table elements
      - Image layout: width, height, border, align on img

   b. Singleton via `sync.Once`: `getEmailPolicy() *bluemonday.Policy`

   c. `substituteCIDImages(html string, attachments []query.AttachmentInfo) string`:
      - Build map: `strings.Trim(att.ContentID, "<>")` -> att.ID (skip empty ContentID)
      - Regex: `(?i)src=["']cid:([^"']+)["']`
      - Replace matches with `src="/attachments/{id}/inline"` when CID found in map
      - Strip unmatched CID refs to `src=""` (avoid broken cid: URLs in browser)

   d. `blockExternalImages(html string) string`:
      - Regex: `(?i)src=["'](https?://[^"']+)["']`
      - Replace with `src=""`
      - This preserves the img tag (and its alt text) but prevents loading

   e. `sanitizeEmailHTML(html string, attachments []query.AttachmentInfo, showImages bool) string`:
      - Step 1: `substituteCIDImages(html, attachments)` — CID before sanitization (bluemonday would strip cid: scheme)
      - Step 2: if !showImages, `blockExternalImages(html)` — block external before sanitization
      - Step 3: `getEmailPolicy().Sanitize(html)` — sanitize with bluemonday

3. Create `internal/web/sanitize_email_test.go` with table-driven tests covering all behaviors listed above. Use the standard Go `testing` package. Import `query` for `AttachmentInfo` type.
