1. **Add messageBody handler** in `internal/web/handlers_messages.go`:

   ```go
   func (h *handlers) messageBody(w http.ResponseWriter, r *http.Request) {
   ```
   - Parse message ID from chi.URLParam(r, "id")
   - Call `h.engine.GetMessage(ctx, id)` to get the full message detail
   - If no BodyHTML, return a minimal HTML page with "No HTML body available" text
   - Parse `showImages` from query param: `r.URL.Query().Get("showImages") == "true"`
   - Call `sanitizeEmailHTML(msg.BodyHTML, msg.Attachments, showImages)`
   - Set response headers:
     - `Content-Type: text/html; charset=utf-8`
     - `Content-Security-Policy: default-src 'none'; script-src 'unsafe-inline'; img-src 'self' data:; style-src 'unsafe-inline' *; font-src *` (when showImages=false, img-src restricts to self+data only, blocking CSS background-image external loads too)
     - When showImages=true, set `img-src * data:` instead
     - Do NOT set X-Frame-Options (this page is meant to be framed)
   - Write a complete standalone HTML document wrapping the sanitized body:
     ```html
     <!DOCTYPE html><html><head><meta charset="utf-8"><style>
     body { margin: 0; padding: 8px; font-family: sans-serif; }
     img[src=""] { display: inline-block; min-width: 16px; min-height: 16px;
       background: #eee; border: 1px dashed #ccc; }
     img[src=""]:after { content: attr(alt); font-size: 12px; color: #666; }
     </style></head><body>
     {sanitized HTML here}
     <script>
     (function(){
       var lastH = 0;
       function report() {
         var h = document.documentElement.scrollHeight;
         if (h !== lastH) { lastH = h; window.parent.postMessage({type:'msgvault-resize',height:h},'*'); }
       }
       report();
       if (window.ResizeObserver) { new ResizeObserver(report).observe(document.body); }
       window.addEventListener('load', report);
       // Debounced MutationObserver for image loads and DOM changes
       var timer;
       if (window.MutationObserver) {
         new MutationObserver(function(){ clearTimeout(timer); timer = setTimeout(report, 50); })
           .observe(document.body, {subtree:true, childList:true, attributes:true});
       }
     })();
     </script></body></html>
     ```
   - The postMessage resize script is the only script in the iframe. bluemonday stripped all email scripts. The iframe sandbox has `allow-scripts` (without `allow-same-origin`) so this resize script runs but the iframe cannot access the parent DOM.

4. **Create `internal/web/backfill.go`**:
   ```go
   // BackfillContentIDs re-parses raw MIME for attachments missing content_id.
   // Runs in a background goroutine, non-blocking on server start.
   func BackfillContentIDs(db *sql.DB, logger *slog.Logger)
   ```
   - Query: `SELECT a.id, a.message_id FROM attachments a WHERE a.content_id IS NULL OR a.content_id = ''`
   - Group by message_id to avoid re-parsing the same message multiple times
   - For each unique message_id:
     - Fetch raw MIME: `SELECT raw_data, compression FROM message_raw WHERE message_id = ?`
     - Decompress if zlib
     - Parse with `mime.Parse(rawData)`
     - For each parsed attachment with non-empty ContentID, update: `UPDATE attachments SET content_id = ? WHERE message_id = ? AND content_hash = ?`
   - Use `golang.org/x/sync/semaphore` or a simple channel to limit concurrency (e.g., 4 goroutines)
   - Log progress: every 100 messages processed, log count
   - Handle errors gracefully: log and continue (don't fail the server)
