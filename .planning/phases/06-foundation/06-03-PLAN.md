           - If groupBy=labels: set filter.Label = filterKey
           - If groupBy=time: set filter.TimeRange.Period = filterKey
         - Propagate sourceId to filter if present
         - If `filterView` is set: sub-aggregate — call `h.engine.SubAggregate(r.Context(), filter, filterViewType, opts)`, render as another aggregate table
         - If `filterView` is NOT set: show messages for this key — call `h.engine.ListMessages(r.Context(), filter)` with pagination, render as messages table
         - Build breadcrumbs: "Aggregate > {viewType} > {filterKey}" (+ "> {filterView}" if sub-aggregating)
         - Render `templates.AggregateDrilldownPage(breadcrumbs, content, ...)`
         - Call `h.renderPage(w, r, filterKey, component)`

       - `func (h *handlers) searchPage(w http.ResponseWriter, r *http.Request)`:
         - Read `q` param from URL query
         - If `q` is empty: render search page with empty results
         - If `q` is non-empty:
           - Parse search query: `parsed := search.Parse(q)`
           - Parse message filter from request (for sourceId, etc.)
           - Set limit to 50, offset from URL params
           - Call `h.engine.SearchFast(r.Context(), parsed, filter, limit, offset)` (fast path: DuckDB metadata search)
           - If fast returns zero results AND `parsed.TextTerms` has entries: fall back to `h.engine.Search(r.Context(), parsed, limit, offset)` (deep FTS5 search)
           - Get total count for pagination: `h.engine.SearchFastCount(r.Context(), parsed, filter)` (for fast results); for deep fallback, count = len(results) (no separate count method for deep)
           - Track which path was used (fast vs deep) to show in UI
         - Build base URL with q param preserved
         - Render `templates.SearchPage(q, results, total, baseURL, searchMode)`
         - Call `h.renderPage(w, r, "Search", component)`

         - The `hx-include="[name='sourceId']"` ensures account filter is included in search requests. Note: this selects the account filter dropdown from the navbar by name attribute.
