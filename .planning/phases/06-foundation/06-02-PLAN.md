    1. Create `internal/web/templates/components.templ`:
       - `templ Pagination(baseURL string, offset, limit int, total int64)`:
         - Shows "Showing {offset+1}-{min(offset+limit, total)} of {FormatNum(total)} messages"
         - Prev link: if offset > 0, render `<a href="..." hx-get="..." hx-select="#main-content" hx-target="#main-content" hx-swap="outerHTML" hx-replace-url="true">Prev</a>`. URL = baseURL with `offset={offset-limit}`.
         - Next link: if `offset+limit < total`, same pattern with `offset={offset+limit}`
         - Use `templ.SafeURL()` for all dynamic href values
       - `templ SortHeader(label, field, currentField, currentDir, baseURL string)`:
         - Renders a clickable `<th>` that sorts by the given field
         - If `field == currentField`: show sort direction arrow (up/down triangle), link reverses direction
         - If `field != currentField`: link sets `sortField={field}&sortDir=desc`
         - Uses `hx-get`, `hx-select="#main-content"`, `hx-target="#main-content"`, `hx-swap="outerHTML"`, `hx-replace-url="true"`

    2. Create `internal/web/handlers_messages.go`:
       - Move `messagesList` and `messageDetail` stubs from handlers.go to this file.
       - `func (h *handlers) messagesList(w http.ResponseWriter, r *http.Request)`:
         - Parse filter using `parseMessageFilter(r)` — this gives pagination (limit 50, offset from URL) and sorting
         - Override default limit to 50 (locked decision: 50 rows/page)
         - Call `h.engine.ListMessages(r.Context(), filter)`
         - Need total count for pagination. Call `h.engine.SearchFastCount(r.Context(), &search.Query{}, filter)` or derive from `AggregateRow.TotalUnique` if available. If the Engine interface doesn't have a clean "count all messages" method, use `SearchFastCount` with an empty query, or call `GetTotalStats` and use `MessageCount`. Prefer `GetTotalStats` with the same sourceId for simplicity — the stat count is the total.
         - Build base URL preserving current query params (sortField, sortDir, sourceId, any filter params) but NOT offset
         - Render `templates.MessagesPage(messages, filter, totalCount, baseURL)`
         - Call `h.renderPage(w, r, "Messages", component)`
       - `func (h *handlers) messageDetail(w http.ResponseWriter, r *http.Request)`:
         - Parse message ID from chi URL param: `chi.URLParam(r, "id")`
         - Call `h.engine.GetMessage(r.Context(), id)`
         - Render `templates.MessageDetailPage(msg)`
         - Call `h.renderPage(w, r, msg.Subject, component)`
         - On not found: `h.renderError(w, r, 404, "Message not found")`

    3. Create `internal/web/templates/messages.templ`:
       - `templ MessagesPage(messages []query.MessageSummary, filter query.MessageFilter, total int64, baseURL string)`:
         - Sort field and direction extracted from filter.Sorting
         - Current sort field as string ("date", "size", "subject") and direction ("asc", "desc")
         - Table with sortable column headers using `SortHeader` component:
           - From (not sortable — no direct sort field)
           - Subject (sortField="subject")
           - Date (sortField="date")
           - Size (sortField="size")
           - Attachments indicator (paper clip icon or "A" count, not sortable)
         - Each row: `<tr data-row data-href="/messages/{id}" class="message-row">` with columns:
           - From: `msg.FromName` if non-empty, else `msg.FromEmail` (truncate at 30 chars)
           - Subject: `msg.Subject` (truncate at 60 chars). Show snippet on hover via title attribute.
           - Date: `FormatDate(msg.SentAt)` or relative time
           - Size: `FormatBytes(msg.SizeEstimate)`
           - Attachments: show count if `msg.HasAttachments`
         - Each row is clickable — links to `/messages/{id}` (used by keyboard nav via data-href)
         - Below table: `@Pagination(baseURL, filter.Pagination.Offset, filter.Pagination.Limit, total)`
         - Empty state if no messages: "No messages found."

    4. Create `internal/web/templates/message.templ`:

           - Subject: `<h2>{ msg.Subject }</h2>`
           - From: formatted addresses (Name <email> or just email)
